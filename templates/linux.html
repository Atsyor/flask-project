Basic Commands in Linux

  1. man shows a manual page for a certain topic.
  2. type man pwd enter to see what the pwd command does.
  3. Use cd ~ to go back to the directory you were in.
  4. ps aux prints details of every process running on your machine.
  5. grep searches the input for lines matching a certain pattern.
  6. Use ls and wc to find out how many files and directories are in your root directory.

    7. Advanced commands

  7. whoami prints out your user ID
  8. ps aux prints details of every process running on your machine
  9. Grep is very powerful. grep ^I sam prints out all the lines in the file sam that start with the character "I"
